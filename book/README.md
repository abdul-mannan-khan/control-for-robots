# Book — Control for Robots

The companion book generated from the 12 lecture HTMLs using the
[kaobook](https://github.com/fmarotta/kaobook) template (LPPL, redistributable).

**Pre-built PDF:** `main.pdf` (257 pages, ~2.3 MB).

## Rebuilding locally

```bash
# requires: texlive-latex-extra, texlive-fonts-extra, texlive-science,
#          texlive-luatex, texlive-bibtex-extra, biber
cd book
lualatex -interaction=nonstopmode main.tex
lualatex -interaction=nonstopmode main.tex   # twice for cross-references
```

## Rebuilding via Docker

The project Dockerfile doesn't ship a full TeX Live by default (it would add ~2 GB).
If you prefer a containerised build, use a TeX image directly:

```bash
docker run --rm -v "$(pwd)":/work -w /work/book \
    texlive/texlive:latest \
    lualatex -interaction=nonstopmode main.tex
```

## Source layout

```
book/
  main.tex                  master document (title, TOC, \input each chapter)
  chapters/
    week_01.tex             one .tex per lecture, auto-generated by pandoc
    ...
    week_12.tex
  images/
    week_01/ ... week_12/   output figures from the simulations
  kaobook.cls               the template class file
  kao*.sty                  companion style packages
  main.bib                  (empty-ish) bibliography
  main.pdf                  the compiled book
```

## Regenerating the LaTeX chapters from source HTML

```bash
python3 /path/to/clean_chapters.py
```

The helper script reads `/home/it-services/auto_control_ws/week_XX/Week*.html`,
runs `pandoc -f html -t latex` per file, strips pandoc-specific boilerplate,
rewrites image paths to `images/week_XX/...`, and wraps each result in a
`\chapter{}` block that kaobook can stitch together. Regenerate and rebuild
after editing any lecture HTML.
