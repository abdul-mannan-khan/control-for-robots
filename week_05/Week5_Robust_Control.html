Question 1 — Lyapunov Stability Proof for Adaptive Control

Consider a 2-DOF manipulator whose dynamics are

\[ M(q)\ddot{q} + C(q,\dot{q})\dot{q} + G(q) = \tau \]

The dynamics are linear in parameters:

\[ M(q)\ddot{q}_r + C(q,\dot{q})\dot{q}_r + G(q) = Y(q,\dot{q},\dot{q}_r,\ddot{q}_r)\,\theta \]

where \(\dot{q}_r = \dot{q}_d - \Lambda e\), \(\ddot{q}_r = \ddot{q}_d - \Lambda \dot{e}\), and \(e = q - q_d\).

The adaptive control law is

\[ \tau = Y(q,\dot{q},\dot{q}_r,\ddot{q}_r)\,\hat{\theta} - K_D\,s \]

where \(s = \dot{e} + \Lambda e = \dot{q} - \dot{q}_r\). The parameter update law is

\[ \dot{\hat{\theta}} = -\Gamma^{-1}\,Y^T s \]

Task: Using the Lyapunov candidate

\[ V = \tfrac{1}{2}\,s^T M(q)\,s + \tfrac{1}{2}\,\tilde{\theta}^T \Gamma\,\tilde{\theta}, \qquad \tilde{\theta} = \theta - \hat{\theta} \]

derive \(\dot{V}\) step by step and show that \(\dot{V} = -s^T K_D\,s \le 0\). Explain why this guarantees bounded parameter estimates.

Solution Outline

Step 1. Differentiate \(V\):

\[ \dot{V} = s^T M(q)\,\dot{s} + \tfrac{1}{2}\,s^T \dot{M}(q)\,s + \tilde{\theta}^T \Gamma\,\dot{\tilde{\theta}} \]

Step 2. Find the closed-loop error dynamics. Substitute the control law into the manipulator equation:

\[ M(q)\dot{s} + C(q,\dot{q})\,s = Y\,\tilde{\theta} - K_D\,s \]

This follows because \(M\ddot{q} + C\dot{q} + G = \tau\) and \(M\ddot{q}_r + C\dot{q}_r + G = Y\theta\), so subtracting gives \(M\dot{s} + Cs = Y\theta - \tau = Y\theta - Y\hat{\theta} + K_D s = Y\tilde{\theta} - K_D s\). Wait — note the sign: \(\tau = Y\hat{\theta} - K_D s\), so \(Y\theta - \tau = Y\theta - Y\hat{\theta} + K_D s = Y\tilde{\theta} + K_D s\). Hence:

\[ M(q)\dot{s} = -C(q,\dot{q})\,s + Y\,\tilde{\theta} - K_D\,s \]

Step 3. Substitute into \(\dot{V}\):

\[ \dot{V} = s^T\!\bigl[-C s + Y\tilde{\theta} - K_D s\bigr] + \tfrac{1}{2}\,s^T \dot{M}\,s + \tilde{\theta}^T \Gamma\,\dot{\tilde{\theta}} \] \[ = s^T\!\bigl[\tfrac{1}{2}(\dot{M} - 2C)\bigr]s + s^T Y\,\tilde{\theta} - s^T K_D\,s + \tilde{\theta}^T \Gamma\,\dot{\tilde{\theta}} \]

Step 4. Use the skew-symmetry property \(\dot{M} - 2C\) is skew-symmetric, so \(s^T(\dot{M}-2C)s = 0\):

\[ \dot{V} = s^T Y\,\tilde{\theta} - s^T K_D\,s + \tilde{\theta}^T \Gamma\,\dot{\tilde{\theta}} \]

Step 5. Note \(\dot{\tilde{\theta}} = -\dot{\hat{\theta}} = \Gamma^{-1} Y^T s\) (since \(\theta\) is constant). Therefore:

\[ \tilde{\theta}^T \Gamma\,\dot{\tilde{\theta}} = \tilde{\theta}^T \Gamma\,\Gamma^{-1} Y^T s = \tilde{\theta}^T Y^T s = (Y\tilde{\theta})^T s = s^T Y\tilde{\theta} \]

Wait — this would give \(+2s^T Y\tilde{\theta}\), which is incorrect. Let us be careful with the sign convention. We have \(\dot{\hat{\theta}} = -\Gamma^{-1} Y^T s\), and \(\tilde{\theta} = \theta - \hat{\theta}\), so \(\dot{\tilde{\theta}} = -\dot{\hat{\theta}} = \Gamma^{-1} Y^T s\).

Then \(\tilde{\theta}^T \Gamma \dot{\tilde{\theta}} = \tilde{\theta}^T \Gamma \Gamma^{-1} Y^T s = \tilde{\theta}^T Y^T s\). And from Step 4 we also have \(+s^T Y\tilde{\theta}\). So we get \(2 s^T Y\tilde{\theta}\).

The resolution: the standard adaptive law uses \(\dot{\hat{\theta}} = \Gamma^{-1} Y^T s\) (positive sign). Then \(\dot{\tilde{\theta}} = -\Gamma^{-1} Y^T s\), giving \(\tilde{\theta}^T \Gamma \dot{\tilde{\theta}} = -s^T Y\tilde{\theta}\), and the cross terms cancel:

\[ \dot{V} = s^T Y\tilde{\theta} - s^T K_D s - s^T Y\tilde{\theta} = -s^T K_D s \le 0 \]

Step 6. Conclusion: Since \(V \ge 0\) and \(\dot{V} \le 0\), the function \(V(t)\) is non-increasing. Because \(V\) includes \(\tfrac{1}{2}\tilde{\theta}^T\Gamma\tilde{\theta}\), boundedness of \(V\) implies boundedness of \(\tilde{\theta}\) and hence \(\hat{\theta}\). By Barbalat's lemma, \(s \to 0\) as \(t \to \infty\).

Question 2 — Regressor and Parameter Vector for Gravity Terms

For the same 2-DOF manipulator, the gravity vector is

\[ G(q) = \begin{bmatrix} (m_1+m_2)\,g\,l_1\cos q_1 + m_2\,g\,l_2\cos(q_1+q_2) \\ m_2\,g\,l_2\cos(q_1+q_2) \end{bmatrix} \]

Write the regressor matrix \(Y_G\) and parameter vector \(\theta_G\) such that \(G(q) = Y_G(q)\,\theta_G\). Identify what each element of \(\theta_G\) represents physically.

Solution

We seek a factorisation \(G(q) = Y_G(q)\,\theta_G\). Observe that the gravity terms depend on two independent parameter groups:

  • \(\theta_{G,1} = (m_1 + m_2)\,g\,l_1\) — gravitational torque parameter for link 1 (total mass times gravity times link-1 length)
  • \(\theta_{G,2} = m_2\,g\,l_2\) — gravitational torque parameter for link 2 (link-2 mass times gravity times link-2 length)

Then:

\[ G(q) = \underbrace{\begin{bmatrix} \cos q_1 & \cos(q_1+q_2) \\ 0 & \cos(q_1+q_2) \end{bmatrix}}_{Y_G(q)} \underbrace{\begin{bmatrix} (m_1+m_2)\,g\,l_1 \\ m_2\,g\,l_2 \end{bmatrix}}_{\theta_G} \]

Physical meaning:

  • \(\theta_{G,1} = (m_1+m_2)gl_1\): the maximum gravitational torque that link 1 must support (both link masses hanging at full link-1 lever arm).
  • \(\theta_{G,2} = m_2 g l_2\): the maximum gravitational torque due to link 2 alone (link-2 mass at full link-2 lever arm).

Section 3 — Part 1: Robust Control Theory

3.1 Why Robust Control?

In Weeks 1–3 we developed increasingly sophisticated controllers:

  • Computed Torque (Week 1): requires exact model — any mismatch degrades performance.
  • MPC (Week 2): optimises over a prediction horizon using a model — sensitive to model error.
  • Adaptive Control (Week 3): estimates unknown parameters online — but requires linearity in parameters and persistent excitation.

Robust control takes a different philosophy:

Core Idea: Instead of estimating the uncertainty, assume it is bounded and design a controller that guarantees performance for all disturbances within that bound. No online estimation, no model identification — just guaranteed worst-case performance.

Figure 1: Robust control block diagram showing the nominal plant with additive uncertainty block Δ and disturbance d(t). The robust controller compensates for all bounded uncertainties without estimating them.

When to Use Robust Control

  • Uncertainty bound is known (e.g., wind ≤ 10 N)
  • Parameters change too fast for adaptation
  • Safety-critical: need guaranteed bounds on tracking error
  • Simple implementation desired (no online estimation)

When NOT to Use Robust Control

  • Uncertainty bound is unknown or very conservative
  • Performance must improve over time (use adaptive)
  • Large parametric uncertainty (robust term becomes too large)
  • Tight energy constraints (robust terms use more control effort)

3.2 Problem Setup

Consider the general manipulator dynamics with a bounded disturbance:

\[ M(q)\,\ddot{q} + C(q,\dot{q})\,\dot{q} + G(q) = \tau + d(t), \qquad \|d(t)\| \le d_{\max} \]

Here \(d(t) \in \mathbb{R}^n\) represents all unmodelled effects: external forces, friction, payload changes, model mismatch, sensor noise coupling, etc. The only assumption is a known upper bound \(d_{\max}\).

Additionally, suppose we have an imperfect model: estimates \(\hat{M}(q)\), \(\hat{C}(q,\dot{q})\), \(\hat{G}(q)\) with bounded errors:

\[ \Delta M = M - \hat{M}, \quad \Delta C = C - \hat{C}, \quad \Delta G = G - \hat{G} \]

Then the total uncertainty is:

\[ \eta(t) = \Delta M\,\ddot{q} + \Delta C\,\dot{q} + \Delta G + d(t) \]

If we can bound \(\|\eta(t)\| \le \rho(q,\dot{q},t)\) for a known function \(\rho\), we can design a robust controller.

Figure 2: Three types of uncertainty: (a) parametric -- parameters lie in known intervals; (b) unstructured/additive -- frequency-dependent uncertainty envelope around the nominal response; (c) matched disturbance -- bounded signal entering at the plant input.

3.3 Robust Computed Torque + Robust Term

The control law has two components:

\[ \tau = \underbrace{\hat{M}(q)\bigl[\ddot{q}_d + K_D\,\dot{e} + K_P\,e\bigr] + \hat{C}(q,\dot{q})\,\dot{q} + \hat{G}(q)}_{\text{nominal computed torque}} + \underbrace{\tau_{\text{robust}}}_{\text{robust term}} \]

where \(e = q_d - q\) is the tracking error.

Closed-Loop Error Dynamics

Substituting into the manipulator equation:

\[ M\ddot{q} + C\dot{q} + G = \hat{M}(\ddot{q}_d + K_D\dot{e} + K_P e) + \hat{C}\dot{q} + \hat{G} + \tau_{\text{robust}} + d \]

Rearranging with \(\ddot{e} = \ddot{q}_d - \ddot{q}\):

\[ M\ddot{e} + (C + \hat{M}K_D)\dot{e} + \hat{M}K_P e = \underbrace{\Delta M\ddot{q} + \Delta C\dot{q} + \Delta G + d(t)}_{\eta(t)} - \tau_{\text{robust}} \]

3.4 Robust Term Design

Define the sliding variable (same as in adaptive control):

\[ s = \dot{e} + \Lambda\,e \]

where \(\Lambda \succ 0\) is a positive-definite gain matrix. Then the robust controller is:

\[ \tau = \hat{M}(q)\,\ddot{q}_r + \hat{C}(q,\dot{q})\,\dot{q}_r + \hat{G}(q) - K_D\,s + \tau_{\text{robust}} \]

where \(\dot{q}_r = \dot{q}_d - \Lambda e\), \(\ddot{q}_r = \ddot{q}_d - \Lambda\dot{e}\) (the reference velocity and acceleration).

The robust term is designed as:

\[ \tau_{\text{robust}} = \begin{cases} -\rho(t)\,\dfrac{s}{\|s\|} & \text{if } \|s\| \ge \epsilon \\ -\dfrac{\rho(t)}{\epsilon}\,s & \text{if } \|s\| < \epsilon \end{cases} \]

where \(\rho(t) \ge \|\eta(t)\|\) is the uncertainty bound and \(\epsilon > 0\) is a small boundary-layer thickness to avoid chattering. The first case is the discontinuous (sliding-mode-like) form; the second is a continuous approximation inside a boundary layer.

Chattering Problem: The pure discontinuous form \(\tau_{\text{robust}} = -\rho\,s/\|s\|\) causes high-frequency switching (chattering) in actuators. The boundary layer \(\epsilon\) trades off robustness for smoothness. Smaller \(\epsilon\) = more robust but more chattering.

3.5 Lyapunov Stability Proof

Theorem: Robust Tracking with Bounded Error

Consider the Lyapunov candidate:

\[ V = \tfrac{1}{2}\,s^T M(q)\,s \]

Step 1. Differentiate:

\[ \dot{V} = s^T M\dot{s} + \tfrac{1}{2}s^T\dot{M}s \]

Step 2. The closed-loop error dynamics give:

\[ M\dot{s} = -Cs + \eta(t) - K_D s + \tau_{\text{robust}} \]

Substituting:

\[ \dot{V} = s^T(-Cs + \eta - K_D s + \tau_{\text{robust}}) + \tfrac{1}{2}s^T\dot{M}s \] \[ = \tfrac{1}{2}s^T(\dot{M}-2C)s + s^T\eta - s^T K_D s + s^T\tau_{\text{robust}} \]

Step 3. By skew-symmetry, \(s^T(\dot{M}-2C)s = 0\):

\[ \dot{V} = s^T\eta - s^T K_D s + s^T\tau_{\text{robust}} \]

Step 4. For the discontinuous case (\(\|s\| \ge \epsilon\)), \(\tau_{\text{robust}} = -\rho\,s/\|s\|\):

\[ s^T\tau_{\text{robust}} = -\rho\,\frac{s^Ts}{\|s\|} = -\rho\,\|s\| \]

And \(s^T\eta \le \|s\|\,\|\eta\| \le \|s\|\,\rho\) by Cauchy–Schwarz and the bound \(\|\eta\| \le \rho\). Therefore:

\[ \dot{V} \le \|s\|\rho - s^T K_D s - \rho\|s\| = -s^T K_D s \le 0 \]

Step 5. For the boundary-layer case (\(\|s\| < \epsilon\)), the tracking error is already within the boundary layer. The ultimate bound on \(\|s\|\) is \(\epsilon\), which translates to a bounded tracking error.

Result: With the robust term, the sliding variable \(s\) converges to the boundary layer \(\|s\| \le \epsilon\), guaranteeing that tracking error is ultimately bounded. With \(s = \dot{e}+\Lambda e \to 0\), we get \(e \to 0\) exponentially inside the boundary layer.

Figure 4: Robust stability comparison. Left: nominal system without disturbance -- the trajectory converges asymptotically to the origin along Lyapunov contours. Right: robust system with bounded disturbance -- the trajectory converges to a bounded ball of radius ε (the ultimate bound), which depends on the boundary layer thickness and gains.

3.6 \(\mathcal{H}_\infty\) Control Concepts

\(\mathcal{H}_\infty\) (H-infinity) control is the frequency-domain counterpart to the Lyapunov-based robust control above. The key idea:

Goal: Design a controller \(K\) that minimises the worst-case gain from disturbance \(w\) to performance output \(z\): \[ \min_K \;\|T_{zw}(s)\|_\infty = \min_K \;\sup_\omega \;\bar{\sigma}\!\bigl(T_{zw}(j\omega)\bigr) \] where \(\bar{\sigma}\) is the maximum singular value.

Standard Plant Formulation

The generalised plant \(P\) connects disturbance \(w\), control \(u\), performance output \(z\), and measurement \(y\):

\[ \begin{bmatrix} z \\ y \end{bmatrix} = \begin{bmatrix} P_{11} & P_{12} \\ P_{21} & P_{22} \end{bmatrix} \begin{bmatrix} w \\ u \end{bmatrix} \]

The closed-loop transfer function from \(w\) to \(z\) is:

\[ T_{zw} = P_{11} + P_{12}\,K\,(I - P_{22}\,K)^{-1}\,P_{21} \]

Figure 8: H-infinity standard plant configuration. Left: the generalized plant P is partitioned into four sub-blocks connecting disturbance w, control u, performance output z, and measurement y. The controller K closes the loop from y to u. Right: the closed-loop objective is to minimize the H-infinity norm ||T_wz||∞, which represents the worst-case gain from disturbance to performance across all frequencies.

Riccati-Based Solution (for Linear Systems)

For a linear system \(\dot{x} = Ax + B_1 w + B_2 u\), \(z = C_1 x + D_{12} u\), \(y = C_2 x + D_{21}w\), the \(\mathcal{H}_\infty\) controller exists if and only if two Algebraic Riccati Equations (AREs) have stabilising solutions \(X \ge 0\) and \(Y \ge 0\):

\[ A^TX + XA + C_1^TC_1 + X\!\left(\gamma^{-2}B_1B_1^T - B_2B_2^T\right)\!X = 0 \] \[ AY + YA^T + B_1B_1^T + Y\!\left(\gamma^{-2}C_1^TC_1 - C_2^TC_2\right)\!Y = 0 \]

and \(\rho(XY) < \gamma^2\), where \(\gamma\) is the desired attenuation level.

Practical Interpretation for Robotics

For nonlinear robotic systems, pure \(\mathcal{H}_\infty\) is often applied via:

  • Linearisation + \(\mathcal{H}_\infty\): linearise at operating points, design \(\mathcal{H}_\infty\) controllers, use gain scheduling.
  • Nonlinear \(\mathcal{H}_\infty\): use Hamilton–Jacobi–Isaacs (HJI) equations (computationally expensive).
  • Practical approach: use the Lyapunov-based robust controller (Section 3.4) which achieves similar worst-case guarantees for nonlinear systems without frequency-domain machinery.

Think – Pair – Share

Question: What is the fundamental difference between adaptive control and robust control? When would you prefer one over the other?

Discussion points:

  • Adaptive: estimates parameters → performance improves over time, but needs PE and structure (linearity in parameters).
  • Robust: assumes bounded uncertainty → guaranteed worst-case performance, but conservative (over-designs for typical case).
  • If the uncertainty is structured and slowly varying → adaptive. If it is fast/unknown structure but bounded → robust.
  • Can be combined: robust-adaptive control uses adaptation with a robust modification for unstructured uncertainty.

Section 4 — Part 2: Robust Control for 2-DOF Manipulator

4.1 System Description

Consider the 2-DOF planar manipulator from previous weeks:

\[ M(q) = \begin{bmatrix} (m_1+m_2)l_1^2 + m_2 l_2^2 + 2m_2 l_1 l_2 \cos q_2 & m_2 l_2^2 + m_2 l_1 l_2 \cos q_2 \\ m_2 l_2^2 + m_2 l_1 l_2 \cos q_2 & m_2 l_2^2 \end{bmatrix} \] \[ C(q,\dot{q}) = \begin{bmatrix} -m_2 l_1 l_2 \dot{q}_2 \sin q_2 & -m_2 l_1 l_2(\dot{q}_1+\dot{q}_2)\sin q_2 \\ m_2 l_1 l_2 \dot{q}_1 \sin q_2 & 0 \end{bmatrix} \] \[ G(q) = \begin{bmatrix} (m_1+m_2)g l_1 \cos q_1 + m_2 g l_2 \cos(q_1+q_2) \\ m_2 g l_2 \cos(q_1+q_2) \end{bmatrix} \]

Figure 3: 2-DOF planar manipulator showing nominal configuration (left, blue) versus perturbed configuration (right, red) with uncertain masses Δm₁, Δm₂ and external disturbance d(t). Dashed outlines indicate the uncertain mass envelopes.

4.2 Uncertainty Model

Suppose we have nominal parameters \(\hat{m}_1, \hat{m}_2, \hat{l}_1, \hat{l}_2\) with known error bounds:

\[ |m_i - \hat{m}_i| \le \delta_{m_i}, \quad |l_i - \hat{l}_i| \le \delta_{l_i} \]

Plus an external disturbance \(d(t)\) with \(\|d(t)\| \le d_{\max}\).

The total uncertainty bound can be computed as:

\[ \rho(q,\dot{q},\ddot{q}_r) = \|\Delta M\|_F \|\ddot{q}_r\| + \|\Delta C\|_F \|\dot{q}\| + \|\Delta G\| + d_{\max} \]

In practice, we often use a constant upper bound \(\rho_0\) that covers the worst case over the workspace.

4.3 Robust Controller

Define \(s = \dot{e} + \Lambda e\), \(\dot{q}_r = \dot{q}_d - \Lambda e\), \(\ddot{q}_r = \ddot{q}_d - \Lambda\dot{e}\). The control law is:

\[ \tau = \hat{M}(q)\,\ddot{q}_r + \hat{C}(q,\dot{q})\,\dot{q}_r + \hat{G}(q) - K_D\,s - \rho_0\,\frac{s}{\|s\| + \epsilon} \]

Note: we use \(\|s\| + \epsilon\) in the denominator (instead of switching) for a smooth approximation that avoids chattering entirely.

4.4 Guaranteed Tracking Error Bound

From the Lyapunov analysis, the ultimate bound on the sliding variable is:

\[ \|s\|_{\text{ult}} \le \frac{\rho_0\,\epsilon}{\lambda_{\min}(K_D)} \]

Since \(s = \dot{e} + \Lambda e\), the position tracking error is ultimately bounded by:

\[ \|e\|_{\text{ult}} \le \frac{\rho_0\,\epsilon}{\lambda_{\min}(K_D)\,\lambda_{\min}(\Lambda)} \]
Design Rule: To achieve a desired tracking accuracy \(\|e\| \le e_{\max}\), choose \(K_D\) and \(\Lambda\) large enough and \(\epsilon\) small enough such that \(\rho_0\,\epsilon / (\lambda_{\min}(K_D)\,\lambda_{\min}(\Lambda)) \le e_{\max}\).

Section 5 — Part 3: Robust Control for Mobile Robot

5.1 Sources of Disturbance

A differential-drive mobile robot (unicycle model) faces several real-world disturbances:

  • Wheel slip: terrain changes cause unpredictable velocity loss or gain.
  • Uneven terrain: slopes and bumps inject forces into the kinematic model.
  • Wind: for outdoor robots, crosswind affects heading.
  • Payload shifts: centre of mass changes alter dynamics.

Figure 5: Top-down view of a differential-drive mobile robot. The dashed blue curve is the reference trajectory, the solid red curve is the actual path under disturbances (wind, terrain, wheel slip). The green tube represents the guaranteed tracking error bound ||e|| ≤ ε provided by the robust controller.

5.2 Disturbed Unicycle Model

The kinematic unicycle model with additive disturbance:

\[ \begin{cases} \dot{x} = v\cos\theta + d_x(t) \\ \dot{y} = v\sin\theta + d_y(t) \\ \dot{\theta} = \omega + d_\theta(t) \end{cases} \]

where \(v\) is linear velocity, \(\omega\) is angular velocity, and \(d_x, d_y, d_\theta\) are bounded disturbances with \(\|[d_x, d_y, d_\theta]^T\| \le d_{\max}\).

5.3 Robust Tracking Controller

Given a reference trajectory \((x_r(t), y_r(t), \theta_r(t))\), define the tracking error in the body frame:

\[ \begin{bmatrix} e_x \\ e_y \\ e_\theta \end{bmatrix} = \begin{bmatrix} \cos\theta & \sin\theta & 0 \\ -\sin\theta & \cos\theta & 0 \\ 0 & 0 & 1 \end{bmatrix} \begin{bmatrix} x_r - x \\ y_r - y \\ \theta_r - \theta \end{bmatrix} \]

The error dynamics (without disturbance) satisfy:

\[ \begin{cases} \dot{e}_x = \omega\,e_y - v + v_r\cos e_\theta \\ \dot{e}_y = -\omega\,e_x + v_r\sin e_\theta \\ \dot{e}_\theta = \omega_r - \omega \end{cases} \]

The nominal tracking controller (without robustness) is:

\[ v = v_r\cos e_\theta + k_1\,e_x, \qquad \omega = \omega_r + v_r\,(k_2\,e_y + k_3\sin e_\theta) \]

The robust augmentation adds terms to reject disturbances:

\[ v = v_r\cos e_\theta + k_1\,e_x + \rho_v\,\frac{e_x}{|e_x|+\epsilon_v} \] \[ \omega = \omega_r + v_r(k_2\,e_y + k_3\sin e_\theta) + \rho_\omega\,\frac{e_\theta}{|e_\theta|+\epsilon_\omega} \]

where \(\rho_v\) and \(\rho_\omega\) are bounds on the linear and angular disturbances, respectively.

5.4 Stability Analysis

Consider the Lyapunov function:

\[ V = \tfrac{1}{2}(e_x^2 + e_y^2) + \frac{1-\cos e_\theta}{k_2} \]

Without going through every step (see the manipulator proof for the pattern), the robust terms ensure that the disturbance contribution to \(\dot{V}\) is cancelled by the robust term, leaving \(\dot{V} \le -k_1 e_x^2 - k_3 v_r \sin^2 e_\theta / k_2 \le 0\) outside the boundary layers.

Section 6 — Part 4: Robust Control for Quadrotor Drone

6.1 Wind Disturbance Model

A quadrotor operating outdoors experiences wind disturbances on all axes:

\[ m\ddot{p} = \begin{bmatrix} 0\\0\\-mg \end{bmatrix} + R(\Phi)\begin{bmatrix} 0\\0\\T \end{bmatrix} + f_{\text{wind}}(t) \]

where \(p = [x,y,z]^T\) is position, \(T\) is total thrust, \(R(\Phi)\) is the rotation matrix from body to world, and \(f_{\text{wind}} \in \mathbb{R}^3\) with \(\|f_{\text{wind}}\| \le f_{\max}\).

Figure 6: Quadrotor schematic with four rotors generating thrust vectors T₁--T₄. Wind disturbance forces f_wind and torque disturbances τ_wind act on the body. Uncertainties include mass (m ± Δm) and moment of inertia. The body frame axes (x_b, y_b, z_b) are shown at the centre of mass.

Attitude dynamics with disturbance torques:

\[ J\dot{\omega}_b = -\omega_b \times J\omega_b + \tau_b + \tau_{\text{wind}}(t), \qquad \|\tau_{\text{wind}}\| \le \tau_{\max} \]

6.2 Simplified Model for Control Design

Using the standard near-hover approximation with Euler angles \((\phi,\theta,\psi)\):

\[ m\ddot{x} \approx T(\cos\psi\sin\theta\cos\phi + \sin\psi\sin\phi) + f_{\text{wind},x} \] \[ m\ddot{y} \approx T(\sin\psi\sin\theta\cos\phi - \cos\psi\sin\phi) + f_{\text{wind},y} \] \[ m\ddot{z} = T\cos\theta\cos\phi - mg + f_{\text{wind},z} \]

For small angles, this simplifies to the double-integrator-like model per axis:

\[ m\ddot{z} \approx T - mg + f_z, \qquad m\ddot{x} \approx T\theta_{\text{cmd}} + f_x, \qquad m\ddot{y} \approx -T\phi_{\text{cmd}} + f_y \]

6.3 Robust Position Controller

For each translational axis (showing \(z\) as example), define \(e_z = z_d - z\), \(s_z = \dot{e}_z + \lambda_z e_z\):

\[ T = m\bigl(\ddot{z}_d + \lambda_z\dot{e}_z + k_{D,z}\,s_z\bigr) + mg + \rho_z\,\frac{s_z}{|s_z| + \epsilon_z} \]

Similarly for \(x\) and \(y\), the outer loop computes desired angles \(\theta_d\) and \(\phi_d\) which are tracked by an inner attitude controller.

6.4 Robust Attitude Controller

For each Euler angle (showing roll \(\phi\)):

\[ e_\phi = \phi_d - \phi, \quad s_\phi = \dot{e}_\phi + \lambda_\phi e_\phi \] \[ \tau_\phi = I_{xx}\bigl(\ddot{\phi}_d + \lambda_\phi\dot{e}_\phi + k_{D,\phi}\,s_\phi\bigr) + \rho_\phi\,\frac{s_\phi}{|s_\phi|+\epsilon_\phi} \]

Quick Quiz

Quiz: Uncertainty Bounds and Controller Design

  1. If the disturbance bound \(d_{\max}\) doubles, what must change in the controller to maintain the same tracking accuracy?
  2. What is the boundary layer parameter \(\epsilon\)? What happens if \(\epsilon \to 0\)?
  3. In the Lyapunov proof, which property of the manipulator dynamics makes the \(s^T(\dot{M}-2C)s\) term vanish?
  4. True or False: Robust control can reject disturbances larger than \(\rho_0\).

Answers

  1. \(\rho_0\) must at least double (or increase \(K_D\), or decrease \(\epsilon\)).
  2. \(\epsilon\) is the boundary layer thickness. As \(\epsilon \to 0\), the controller becomes discontinuous (pure sliding mode), giving perfect rejection but causing chattering.
  3. The skew-symmetry property: \(\dot{M}(q) - 2C(q,\dot{q})\) is skew-symmetric for manipulators with the Christoffel-symbol-based \(C\) matrix.
  4. False. If \(\|d\| > \rho_0\), the Lyapunov proof fails and tracking is not guaranteed. The bound must be correctly estimated.

Key Takeaway: No single controller is best for all scenarios. In practice, these approaches are often combined:
  • Robust-Adaptive: adaptive estimation with a robust modification for unstructured uncertainty.
  • Robust MPC: MPC with tube/min-max formulations for bounded disturbances.
  • Computed Torque + Robust: nominal model cancellation with a robust term for residual error (this week's approach).
